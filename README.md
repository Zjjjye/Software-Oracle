# 🔮 Software Oracle
Decode the Engineering "Fate" of GitHub Repositories with AI and Mysticism.
Software Oracle is an experimental full-stack application that bridges the gap between modern software engineering metrics and ancient Eastern philosophy. By analyzing GitHub metadata (commits, issues, stars, and structure), it uses DeepSeek-V3 AI to map a project's "Engineering DNA" into the Five Elements (Wu Xing) model, visualized through an interactive 3D Spiritual Sphere.

# 🚀Get Started

Follow these steps to run the Oracle on your local machine.

1. **Prerequisites**

Ensure you have Node.js (v18+) installed.

2. **Installation**

```
git clone https://github.com/Zjjjye/Software-Oracle.git
cd Software-Oracle
npm install
```

4. **Environment Variables**

   Create a .env file in the root directory and add your API keys:
```
   DEEPSEEK_API_KEY=your_deepseek_api_key_here
   GITHUB_TOKEN=your_github_personal_access_token
```

4. **Development**

```

   Run the frontend (Vite)
   npm run dev·
   
   Run the backend server
   node api/index.js
   
```
